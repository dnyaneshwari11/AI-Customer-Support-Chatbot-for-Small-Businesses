# AI-Customer-Support-Chatbot-for-Small-Businesses
AI Customer Support Chatbot for Small Businesses is a web-based application that helps businesses handle customer queries efficiently. It works as a virtual assistant, answering common questions, sharing product details, and providing real-time support. Built with Flask and AI, it ensures 24/7 availability and improves customer experience. overall!
